# Python LLM

Can we use AWS Bedrock to use LLMs in Python?

## Setup

```sh
mise install
pdm install # using pdm for bants
```

## Login to AWS

```sh
assume
```

## Run scripts

```sh
python examples/bedrock_boto3.py
```

```sh
python examples/bedrock_langchain.py
python examples/bedrock_langchain_structured.py
```
